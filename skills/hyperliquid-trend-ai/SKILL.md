# Hyperliquid AI趋势策略

## 简介

这个策略让AI自动在Hyperliquid交易所做趋势跟踪交易。分析比特币、以太坊、SOL的价格走势，找到好的买卖点自动下单。

**核心特点：**
- 双时间框架确认：15分钟找信号，1小时定方向
- 智能评分系统：满分100分，70分以上才开仓
- 自动风控：止损止盈全自动，不用盯盘
- 严格限制：最多同时持有2个币，每天最多交易3次

## 使用前准备

1. 安装Hyperliquid插件：`npx skills add okx/plugin-store --skill hyperliquid-plugin`
2. 账户里要有USDC作为保证金
3. 设置风险参数：默认5倍杠杆，每次用20%资金
4. 模拟交易：加 `--dry-run` 参数，不真下单先测试

## 使用命令

### 分析市场

```bash
hyperliquid-trend-ai 分析 --币种 BTC --周期 15分钟
```

**什么时候用**：想看现在能不能买/卖
**输出结果**：评分（0-100分）、方向（做多/做空/观望）、建议入场价、止损价、止盈价
**例子**：`hyperliquid-trend-ai 分析 --币种 ETH`

### 执行交易

```bash
hyperliquid-trend-ai 交易 --币种 BTC --方向 做多 --数量 0.05 --杠杆 5 --策略ID hyperliquid-trend-ai
```

**什么时候用**：分析显示评分≥70分，决定开仓
**重要**：必须加 `--策略ID hyperliquid-trend-ai`，否则不算比赛成绩

### 查看持仓

```bash
hyperliquid-trend-ai 监控 --显示全部
```

**什么时候用**：想看现在持仓盈亏情况
**输出结果**：每个币的盈亏、距离止损还有多远、是否已移保本

### 平仓

```bash
hyperliquid-trend-ai 平仓 --币种 BTC --原因 止盈
```

**什么时候用**：想手动平仓，或策略自动触发平仓

## 策略逻辑（AI怎么决策）

### 四维度评分系统

**1. 趋势强弱（占40分）**
- 价格 > 短期均线 > 长期均线：+40分（强势上涨）
- 价格 > 短期均线：+20分（一般上涨）
- 价格 < 短期均线 < 长期均线：-40分（强势下跌，做空信号）

**2. 动量确认（占25分）**
- RSI 30-40：+15分（超卖反弹）
- RSI < 30：+25分（深度超卖，高概率反弹）
- RSI 40-60：0分（震荡，无方向）
- RSI 60-70：+15分（强势延续）

**3. 突破信号（占30分）**
- 价格突破最近6根K线高点0.3%，且成交量放大20%：+30分
- 价格跌破最近6根K线低点0.3%，且成交量放大20%：-30分

**4. 波动率过滤（占5分）**
- 波动率0.5%-3%：+5分（正常波动）
- 波动率>3%：-5分（波动太大，减小仓位）
- 波动率<0.3%：-10分（没波动，跳过）

### 开仓条件（必须同时满足）
1. 总分≥70分
2. 1小时趋势和15分钟信号同向（不做逆势单）
3. 成交量确认突破（放量1.2倍以上）
4. 当前持仓<2个币
5. 今日交易<3次

### 风险管理
- **仓位大小**：每次用20%资金（波动大时自动减到10%）
- **止损**：入场价反向2%自动止损
- **止盈**：入场价正向6%自动止盈
- **移保本**：盈利3%时，止损移到入场价（保证不亏）
- **移动止盈**：盈利5%后，价格回撤3%自动平仓
- **日限制**：每天最多3笔交易，做完就停

## 实际例子

### 例子1：比特币做多

1. 分析：`hyperliquid-trend-ai 分析 --币种 BTC`
2. 结果：评分75分，方向做多，建议入场价67200，止损65856，止盈71232
3. 确认1小时趋势也是上涨
4. 下单：`hyperliquid-trend-ai 交易 --币种 BTC --方向 做多 --数量 0.05`
5. 自动监控，等止盈或止损

### 例子2：以太坊做空

1. 分析显示：评分-72分，方向做空
2. 1小时趋势也是下跌
3. 下单：`hyperliquid-trend-ai 交易 --币种 ETH --方向 做空 --数量 0.5`
4. 止损设在+2%，止盈设在-6%

## 常见问题

| 问题 | 原因 | 解决办法 |
|------|------|---------|
| "评分太低" | 市场没明显趋势 | 等30分钟再分析 |
| "已达持仓上限" | 已有2个持仓 | 先平一个或等 |
| "今日交易已满" | 今天已做3笔 | 明天再做 |
| "趋势不一致" | 1小时和15分钟方向相反 | 跳过，不做逆势单 |
| "余额不足" | USDC不够开仓 | 充值或减少数量 |
| "API超时" | 网络慢 | 等10秒重试 |

## 风险提示

- **风险等级**：高级 - 本策略会自动执行真实交易
- **默认模式**：模拟模式（dry-run），实盘需关闭
- **最大杠杆**：5倍（可调，但不建议超过10倍）
- **止损**：强制开启，不能关闭，默认2%
- **免责声明**：过往业绩不代表未来表现，加密货币交易可能导致本金全部亏损
